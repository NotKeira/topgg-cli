# Contributing

Thanks for contributing.

This project is early-stage, but we still expect disciplined changes and clear pull requests.

## Development Principles

- Keep user-facing copy in British English.
- Keep API integrations aligned with documented top.gg routes.
- Add or update tests for every meaningful behavioural change.
- Prefer small, focused pull requests over broad rewrites.

## Repository State

This repository is currently in scaffold mode.
Runtime code is intentionally absent at this stage.

As implementation lands, this guide will grow with concrete setup, test, and release instructions.

## Pull Requests

1. Create a branch from `main`.
2. Ensure CI passes.
3. Update `CHANGELOG.md` when behaviour or policy changes.
4. Include tests for your change.
