# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |
| < 0.1.0 | No        |

## Reporting a Vulnerability

Please report vulnerabilities privately.

1. Email: `security@keirahopkins.co.uk`.
2. Include a clear reproduction path, expected impact, and any suggested remediation.
3. You should receive an acknowledgement within 3 working days.
4. We aim to ship a fix or mitigation for high-severity issues within 14 days.

Please do not open public issues for undisclosed vulnerabilities.
